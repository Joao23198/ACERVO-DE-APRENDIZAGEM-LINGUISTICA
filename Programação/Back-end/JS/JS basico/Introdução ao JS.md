1. O que é JavaScript
Linguagem de programação criada em 1995 por Brendan Eich.

Usada para adicionar interatividade às páginas web.

Funciona tanto no navegador (front‑end) quanto no servidor (Node.js).

2. História
Inicialmente chamada LiveScript.

Padronizada como ECMAScript em 1997.

Evoluiu com versões importantes:

ES5 (2009) → trouxe melhorias de sintaxe.

ES6/ES2015 (2015) → adicionou let, const, arrow functions, classes e módulos.

3. Como funciona
É uma linguagem interpretada: o navegador executa o código linha por linha.

Pode ser escrita:

Dentro de arquivos HTML usando <script>.

Em arquivos .js separados.

Exemplo simples:

html
<!DOCTYPE html>
<html>
  <body>
    <h1>Bem-vindo!</h1>
    <script>
      alert('Olá, mundo!');
    </script>
  </body>
</html>
4. Primeiros conceitos
Variáveis: armazenam dados (let nome = "João";).

Funções: blocos de código que executam ações.

Eventos: respostas a ações do usuário (ex.: clique em botão).

5. Aplicações práticas
Validação de formulários.

Atualização de conteúdo sem recarregar a página.

Criação de menus interativos.

Exibição de animações e gráficos dinâmicos.
