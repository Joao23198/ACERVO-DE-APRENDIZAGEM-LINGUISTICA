O que é HTML?

HTML não é uma linguagem de programação, mas sim de marcação. Serve para estruturar o conteúdo de páginas web e é interpretado pelos navegadores (Chrome, Firefox, Edge).

HTML (Hypertex Markup Language) é a base de todos os sites, que iniciam-se com HTML e tem a função de estruturar informações variadas, como: Títulos, parágrafos, listas, tabelas e formulários.

exemplo simples:

<!DOCTYPE html>
<html lang="pt-BR">
  <head>
    <meta charset="UTF-8">
    <title>Exemplo HTML</title>
  </head>
  <body>
    <h1>Olá, Mundo!</h1>
    <p>Este é meu primeiro parágrafo em HTML.</p>
    <a href="https://www.exemplo.com">Visite meu site</a>
  </body>
</html>
